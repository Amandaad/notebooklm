# 📘 Miniguia de Estudos com NotebookLM + Inteligência Artificial

## 🎯 Contexto e Objetivos

Este projeto tem como objetivo explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa, utilizando o NotebookLM para organizar, analisar e consolidar conhecimento a partir de múltiplas fontes.

### 📌 Contexto
Com o crescimento das ferramentas de IA, torna-se essencial aprender não apenas a consumir respostas, mas a estruturar perguntas, validar informações e construir conhecimento de forma crítica.

### 🎯 Objetivos
- Entender como a IA pode apoiar o processo de aprendizagem
- Praticar engenharia de prompts
- Desenvolver pensamento crítico sobre respostas geradas por IA
- Criar um material de estudo reutilizável
- Consolidar conhecimento em formato estruturado

### 🧠 Critérios de seleção:
- Fontes confiáveis (acadêmicas e institucionais)
- Relevância para aprendizagem e IA
- Conteúdo atualizado
- Diversidade de perspectivas
